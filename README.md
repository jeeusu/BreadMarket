# 🥐 BreadMarket

이 프로젝트는 **JSP**를 활용한 온라인 **빵 쇼핑몰**입니다.

---

## 📦 사용 기술

- Java 17  
- MySQL  
- JSP / Servlet  
- JSTL  

---

## 🛠 주요 기능

- 회원가입 및 로그인
- 상품 목록 보기
- 장바구니 담기
- 관리자 상품 등록, 삭제 및 관리
